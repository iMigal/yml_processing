Реализация агрегатора товаров используя YML файлы сторонних магазинов
=============================

ПО
----------
Тест проходил на денвере.
PHP 5.3
MySQL 5.5.25


ИНСТАЛЯЦИЯ
----------
0. Распакуйте архив.
1. Скопируйте все файлы и каталоги в корневую папку.
2. Импортируйте файла db.sql в базу данных.
3. В классе database (путь: 'www/classes/database.php') измените параметры подключения к базе данных.
